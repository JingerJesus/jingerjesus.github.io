Breaking Down the Hardware:

The Tower of Babel's hardware can be split up into four main sections: Data Processing, The Stack, Control Logic, and Memory.

Data Processing:

The data processing portion of the computer consists of 4 all-purpose registers, the Arithmetic Logic Unit (ALU), a Flag Register, and two Output registers.

The four all-purpose registers, labelled A-D respectively, are used for temporary data storage and allows for data manipulation via the ALU.
