Breaking Down the Hardware

The Tower of Babel's hardware can be split up into four main sections: Data Processing, The Stack, Control Logic, and Memory.

Data Processing:

The data processing portion of the computer consists of 4 all-purpose registers, the Arithmetic Logic Unit (ALU), a Flag Register, and two Output registers.

The four all-purpose registers, labelled A-D respectively, are used for temporary data storage and allows for data manipulation via the ALU. The vast majority of operations run entirely within these few components.

Math operations and basic boolean logic happens within the ALU, however the four all-purpose registers help to feed data in and allow for anything to happen
